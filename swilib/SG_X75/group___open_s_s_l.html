OpenSSL

Detailed Description

Built-in OpenSSL.

See openssl docs for more info.

Usage: #include <swilib.h> or #include <swilib/openssl.h>

Also, you can use standart openssl headers with #include <builtin/openssl/...> Alternatively, you can add -Ipath/to/include/builtin to the compiler flags.

Note
For GCC: These functions is available only when define -DSWILIB_OPENSSL is specified in compiler flags.
Alternatively, you can include these functions directly by #include <swilib/openssl.h>
