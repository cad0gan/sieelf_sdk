PNG

Detailed Description

Built-in libpng.

See libpng docs for more info.

Usage: #include <swilib.h> or #include <swilib/png.h>

Note
For GCC: These functions is available only when define -DSWILIB_LIBPNG is specified in compiler flags.
Alternatively, you can include these functions directly by #include <swilib/png.h>
