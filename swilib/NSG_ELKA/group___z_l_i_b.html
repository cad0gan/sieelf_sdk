ZLIB

Detailed Description

Built-in zlib functions.

See zlib 1.2.1 docs for more information.

Usage: #include <swilib.h> or #include <swilib/zlib.h>

Note
For GCC: These functions is available only when define -DSWILIB_ZLIB is specified in compiler flags.
Alternatively, you can include these functions directly by #include <swilib/zlib.h>
